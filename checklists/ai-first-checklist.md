# AI-FIRST Marketing Checklist

Checklist para garantir que todas as decisões e processos do squad Fábrica de Receita V4
seguem a filosofia AI-FIRST da V4 Company (Mkt.Lab).

## Antes de Qualquer Iniciativa de Marketing

### Dados
- [ ] Existe tracking adequado para medir o resultado?
- [ ] Os dados necessários estão centralizados e acessíveis?
- [ ] Há baseline para comparação (antes vs depois)?
- [ ] Os dados são limpos, sem anomalias ou gaps?

### IA como Primeiro Recurso
- [ ] Existe uma ferramenta de IA que pode executar isso melhor que humano?
- [ ] A IA pode escalar o que o humano faz manualmente?
- [ ] Foi considerado o uso de automação antes de contratar/alocar humano?
- [ ] Os modelos preditivos disponíveis foram consultados?

### Hipótese e Experimento
- [ ] A iniciativa tem hipótese clara e testável?
- [ ] Existe critério de sucesso mensurávelanteriormente definido?
- [ ] O tamanho de amostra e duração do teste foram calculados?
- [ ] Há grupo de controle quando necessário?

## Durante a Execução

### Tráfego AI-First
- [ ] Smart bidding ativado (tCPA/tROAS/Maximize Conversions)?
- [ ] Advantage+ / Performance Max ativo com assets corretos?
- [ ] Lookalike audiences baseadas em clientes de alto LTV?
- [ ] Dynamic creative optimization habilitado?
- [ ] Exclusão de audiências de baixa qualidade configurada?

### Engajamento AI-First
- [ ] Segmentação dinâmica por comportamento ativa?
- [ ] Send-time optimization no email configurado?
- [ ] Conteúdo personalizado por estágio do funil?
- [ ] Lead scoring automático rodando?
- [ ] Trigger-based automations ativas?

### Conversão AI-First
- [ ] Propensity scoring identificando leads quentes?
- [ ] Dynamic content/CTA por perfil de visitante?
- [ ] Chatbot de qualificação ativo 24/7?
- [ ] Abandonment recovery automático configurado?
- [ ] Testes A/B com análise estatística automática?

### Retenção AI-First
- [ ] Churn prediction model monitorando a base?
- [ ] Health score calculado e atualizado em tempo real?
- [ ] Alertas de risco de churn configurados?
- [ ] Next Best Action automático para CS?
- [ ] Upsell triggers por comportamento de produto?

## Validação de ROI

### Antes de Escalar
- [ ] ROAS está acima do mínimo aceitável?
- [ ] CAC está dentro do LTV:CAC ratio saudável (>3:1)?
- [ ] Payback period está dentro do aceitável (<12 meses)?
- [ ] Os resultados são estatisticamente significativos (>95%)?
- [ ] A escala foi testada incremetalmente antes de multiplicar?

### Após 30 Dias
- [ ] Os resultados confirmam a hipótese original?
- [ ] O ROI está dentro ou acima da projeção?
- [ ] Há sinais de saturação ou fadiga?
- [ ] O experimento deve ser documentado no knowledge base?

## Governança de Dados

- [ ] Todos os dados de clientes estão sendo coletados com consentimento (LGPD)?
- [ ] Os dados estão centralizados no CDP/CRM?
- [ ] Há backup e disaster recovery para dados de marketing?
- [ ] Os modelos de ML estão sendo retrainados regularmente?
- [ ] Os dados de terceiros (3rd party) foram substituídos por 1st party data?
