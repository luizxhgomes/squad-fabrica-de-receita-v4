# Checklist: Validar Diagnóstico

## Uso
Executar este checklist ANTES do Board 1 para garantir que o diagnóstico está robusto e pode ser apresentado ao cliente com confiança.

**Agente:** dr-chief ou diagnosticador
**Quando:** D12-D13 (antes do Board 1 em D14)

---

## Seção 1: Qualidade dos Dados (4 itens)

- [ ] **1.1** Dados do funil obtidos de fonte primária (CRM, Analytics, Ads Manager): não estimativas do cliente
- [ ] **1.2** Período de análise consistente (mínimo 90 dias de dados históricos)
- [ ] **1.3** Dados de pelo menos 2 etapas do funil confirmados com evidência objetiva
- [ ] **1.4** Ticket médio e churn rate calculados com dados reais (não declarado pelo cliente)

## Seção 2: Identificação da Trava (4 itens)

- [ ] **2.1** Funil mapeado com taxas de conversão em cada etapa (Exposição → Lead → SQL → Reunião → Cliente)
- [ ] **2.2** Etapa com maior perda percentual identificada e documentada
- [ ] **2.3** Trava candidata mapeada corretamente para a etapa (conforme tabela T1-T8)
- [ ] **2.4** Alternativas descartadas com justificativa (por que NÃO é outra trava)

## Seção 3: 3 Testes de Confirmação (3 itens)

- [ ] **3.1** Teste 1 (Evidência Direta) aplicado e pontuado com evidência objetiva
- [ ] **3.2** Teste 2 (Causa-Raiz) aplicado: pelo menos 2 EIs conectados como consequência
- [ ] **3.3** Teste 3 (Custo de Inação) calculado com número real (R$/mês perdido)

**Pontuação mínima para GO:** 4/6 pts

## Seção 4: CRT (3 itens)

- [ ] **4.1** Efeito Indesejável Central (EIC) documentado em linguagem do cliente
- [ ] **4.2** Mínimo 3 EIs secundários conectados ao EIC com relação de causalidade
- [ ] **4.3** Mínimo 3 causas-raiz da trava identificadas

## Seção 5: Subordinação (2 itens)

- [ ] **5.1** Travas secundárias identificadas como consequência (não atacar)
- [ ] **5.2** Justificativa documentada para cada trava descartada

## Seção 6: Artefato Inicial (2 itens)

- [ ] **6.1** Artefato inicial selecionado com base na trava confirmada
- [ ] **6.2** Proposta do artefato preparada com métricas de sucesso e prazo de implementação

## Seção 7: Materiais do Board 1 (2 itens)

- [ ] **7.1** Slides/apresentação do diagnóstico preparados
- [ ] **7.2** Template de ata do Board 1 preparado para assinatura
