# Tech Stack: Squad Fábrica de Receita

## Princípios de Stack

1. **Integração antes de mais ferramentas**: antes de adicionar nova ferramenta, verifique se as existentes se integram
2. **1st party data primeiro**: minimize dependência de dados de terceiros
3. **API-first**: prefira ferramentas com API robusta para automações
4. **Custo x ROI**: cada ferramenta precisa se pagar com o ROI que gera
